2017 - Open Source Vlix Http Web Server

Open Source Nuget Package
C# .NET
Around 2017, I developed a simple Static Content Web Server for the purpose of embedding inside Open-Plant. At that time there was no embeddable Http Server for dotnet framework. Most IoT Platforms had to use 3rd party web servers such as IIS and Apache. This project gave me a deep understanding about the web and TCP/IP. The project was made open source and published in Nuget. It has been download 2000 times.
